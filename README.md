# 🚦⚡ AI Smart Traffic & Piezoelectric Energy System

An end-to-end **Smart City solution** that combines **AI-based traffic optimization** and **piezoelectric energy generation** to improve urban mobility and sustainability.

---

## 🌟 Overview

This project integrates:

* 🚗 **Computer Vision (YOLOv8)** for vehicle detection
* 🚦 **AI Traffic Control** for dynamic signal optimization
* ⚡ **Piezoelectric Energy System** for renewable energy generation
* 📊 **Interactive Dashboards** for monitoring and analysis

---

## 🚀 Features

### 🚦 Smart Traffic Control

* AI-based traffic signal allocation
* Real-time vehicle density analysis
* Emergency vehicle prioritization 🚑
* Fairness-based signal distribution

---

### 🤖 YOLO Vehicle Detection

* Real-time vehicle detection using YOLOv8
* Works with video input or camera feed
* Accurate traffic density estimation

---

### ⚡ Piezoelectric Energy System

* Converts vehicle pressure into electrical energy
* Calculates:

  * Energy generated ⚡
  * CO₂ reduction 🌱
  * Revenue potential 💰
* Comparison with traditional systems

---

### 📊 Dashboards (Streamlit)

* 🚦 Traffic Dashboard
* ⚡ Piezo Energy Dashboard
* Clean, interactive, and screenshot-friendly UI

---

## 🧠 System Architecture

```
YOLO Detection → Traffic Density → AI Signal Control → Dashboard  
                          ↓  
                  Piezo Energy Model → Sustainability Metrics
```

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 📊
* OpenCV 🎥
* YOLOv8 (Ultralytics) 🤖
* Matplotlib 📈

---

## ▶️ How to Run

### 1️⃣ Install dependencies

```bash
pip install streamlit matplotlib opencv-python ultralytics
```

---

### 2️⃣ Run YOLO Traffic Detection

```bash
python yolo_traffic.py
```

---

### 3️⃣ Run Traffic Dashboard

```bash
streamlit run traffic_dashboard.py
```

---

### 4️⃣ Run Piezo Dashboard

```bash
streamlit run piezo_dashboard.py
```
